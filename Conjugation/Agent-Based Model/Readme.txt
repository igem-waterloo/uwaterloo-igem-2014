This program is written in Python 2.7.8 and requires the following packages:
-matplotlib
-numpy
-pyparsin
-dateutil
-pytz
-six

These can all be found at http://www.lfd.uci.edu/~gohlke/pythonlibs/

grid_abm: The main script to run. This includes all of the development, objects, structure, etc.
data_plot: The script to create a scatterplot with the total populations over time.
hex_plot: The script to draw pictures every time step. This contains the shapes, colours, borders, etc.
